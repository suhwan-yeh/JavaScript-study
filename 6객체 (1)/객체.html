0. 객체 기본 개념과 메모리 구조

1. 객체(Object)란?

객체는 여러 개의 값을 하나의 참조값으로 관리하는 참조형 데이터입니다. 객체 안에는 속성(Property)과 메소드(Method)를 작성할 수 있습니다.

용어정확한 의미
객체(Object)여러 값을 하나의 구조로 묶어서 관리하는 참조형 데이터
속성(Property)객체 안에 저장된 데이터
메소드(Method)객체 안에 속성값으로 저장된 함수
key속성 이름
value속성에 저장된 값
참조값(Reference)객체가 생성된 메모리 위치를 가리키는 값
작성 문법 - 객체 리터럴 기본 구조
let 객체참조변수 = {
    속성명: 값,
    메소드명: function(){
        실행할 코드
    }
};
코드 예제 - 속성과 메소드 구분
let user = {
    name: "홍길동",
    age: 20,
    printInfo: function(){
        console.log(this.name);
        console.log(this.age);
    }
};

user.name;        // 속성 접근
user.printInfo(); // 메소드 호출
정리
객체 안에 일반 값이 저장되면 속성입니다. 객체 안에 함수가 저장되면 메소드입니다. 메소드는 함수와 다른 종류의 문법이 아니라, 객체의 속성값으로 들어간 함수입니다.

2. 객체와 메모리

객체도 메모리에 생성됩니다. 다만 변수에는 객체 전체가 직접 들어가는 것이 아니라, 객체를 찾을 수 있는 참조값이 저장됩니다.

작성 문법 - 객체 참조값 저장 구조
let user = {
    name: "홍길동"
};
객체 생성{ name: "홍길동" } 객체가 메모리에 생성됩니다.
참조값 저장user 변수에는 객체를 가리키는 참조값이 저장됩니다.
속성 접근user.name은 참조값을 통해 객체에 접근한 뒤 name 속성값을 읽습니다.
코드 예제 - 참조형 데이터 확인
let user1 = { name: "홍길동" };
let user2 = user1;

user2.name = "김자바";

console.log(user1.name); // "김자바"
console.log(user2.name); // "김자바"
주의
객체를 다른 변수에 대입하면 객체 자체가 새로 복사되는 것이 아니라 같은 객체를 가리키는 참조값이 복사됩니다.

2. 객체의 속성 접근, 수정, 추가, 삭제

1. 점 표기법

점 표기법은 객체의 속성명이나 메소드명이 코드에 직접 작성되어 있을 때 사용합니다.

작성 문법 - 점 표기법으로 속성 사용
객체참조변수.속성명
객체참조변수.속성명 = 새값
객체참조변수.새속성명 = 값
객체참조변수.메소드명()
코드 예제 - 점 표기법 사용
let book = {
    title: "JavaScript",
    price: 30000
};

console.log(book.title); // 속성값 읽기
book.price = 35000;      // 속성값 수정
book.writer = "홍길동";  // 새 속성 추가

console.log(book);

2. 대괄호 표기법

대괄호 표기법은 속성명을 문자열로 작성할 때 사용합니다. 속성명이 변수에 저장되어 있거나, 속성명에 특수 문자가 포함될 때 사용할 수 있습니다.

작성 문법 - 대괄호 표기법으로 속성 사용
객체참조변수["속성명"]
객체참조변수["속성명"] = 새값
객체참조변수[변수명]
코드 예제 - 대괄호 표기법 사용
let book = {
    title: "JavaScript",
    price: 30000
};

console.log(book["title"]);

let key = "price";
console.log(book[key]);

3. 객체 안에 메소드 만들기와 this 이해하기

1. 객체 안에 메소드 작성하기

함수는 값처럼 변수나 속성에 저장될 수 있습니다. 함수가 객체의 속성값으로 저장되면 그 함수를 메소드라고 부릅니다.

작성 문법 - 메소드 작성과 호출
let 객체참조변수 = {
    속성명: 값,
    메소드명: function(){
        실행할 코드
    }
};

객체참조변수.메소드명();
코드 예제 - 메소드 호출
let printer = {
    message: "출력할 문자열",
    print: function(){
        console.log(this.message);
    }
};

printer.print();
함수와 메소드 구분
function print(){}처럼 독립적으로 선언되면 함수입니다. 객체.print = function(){}처럼 객체 안에 저장되면 메소드입니다.

2. this의 의미

this는 메소드가 실행될 때, 그 메소드를 호출한 객체를 가리키는 키워드입니다. 객체 내부의 메소드에서 같은 객체의 속성에 접근할 때 사용합니다.

작성 문법 - this 사용 문법
let 객체참조변수 = {
    속성명: 값,
    메소드명: function(){
        this.속성명
    }
};
코드 예제 - this로 같은 객체의 속성 접근
let book = {
    title: "JavaScript",
    done: false,
    finish: function(){
        if(this.done === false){
            console.log(this.title + " 읽는 중");
        }else{
            console.log(this.title + " 완독");
        }
    }
};

book.finish();
위치작성 방식의미
객체 밖에서 속성 접근book.donebook 참조변수를 통해 속성 접근
객체 안의 메소드에서 속성 접근this.done현재 메소드를 호출한 객체의 속성 접근

1. 생성자 함수

생성자 함수는 같은 속성 구조를 가진 객체를 여러 개 생성하기 위한 함수입니다. new 키워드와 함께 호출합니다.

작성 문법 - 생성자 함수 작성
function 생성자함수명(매개변수1, 매개변수2){
    this.속성명1 = 매개변수1;

function Member(id, name){

    };
}

let m1 = new Member("user01", "홍길동");
let m2 = new Member("user02", "김자바");

m1.print();
m2.print();

3. 원기둥 객체 실습

반지름과 높이를 속성으로 저장하고, 부피를 계산하는 기능을 메소드로 작성합니다.

작성 문법 - 계산 기능을 메소드로 작성
class Cylinder {
    constructor(radius, height){
        this.radius = radius;
        this.height = height;
    }

    getVolume(){
        return this.radius * this.radius * Math.PI * this.height;
    }
}
코드 예제 - 원기둥 부피 계산
let c1 = new Cylinder(5, 10);
console.log(c1.getVolume());
