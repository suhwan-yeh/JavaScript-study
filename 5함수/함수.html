1. 함수란?

개념 정의
함수는 실행할 코드를 하나의 이름으로 선언해 두고, 필요할 때 그 이름으로 호출해서 실행하는 문법입니다.
핵심 흐름
  1. 함수를 선언하면 실행할 코드가 준비됩니다.
  2. 함수는 선언만 해서는 실행되지 않습니다.
  3. 함수 이름 뒤에 ()를 붙여 호출해야 실행됩니다.
  4. 함수는 값을 받을 수 있습니다. 이때 사용하는 변수가 매개변수입니다.
  5. 함수는 실행 결과를 함수 밖으로 돌려줄 수 있습니다. 이때 사용하는 문법이 return입니다.
작성 문법 
function 함수명() {
  실행할 코드;
}

함수명();
코드 읽는 순서 
function gugudan() { ... }
위 코드는 gugudan 함수 선언입니다. 실행할 코드가 준비됩니다. 
gugudan();
위 코드는 gugudan 함수 호출입니다. 준비된 함수 내부 코드가 실행됩니다.
실습: gugudan_console.html

2. 함수 선언문

개념 정의
함수 선언문은 function 키워드와 함수명을 사용해서 함수를 선언하는 방식입니다.
작성 문법 
function 함수명(매개변수1, 매개변수2) {
  실행할 코드;
  return 반환값;
}

함수명(인수1, 인수2);
예제 코드 읽기 
function add(a, b) {
  return a + b;
}

let result = add(3, 5);
console.log(result);
  1. add 함수가 선언됩니다.
  2. a, b는 매개변수입니다.
  3. add(3, 5)를 만나면 함수가 호출됩니다.
  4. 인수 3은 매개변수 a에 저장됩니다.
  5. 인수 5는 매개변수 b에 저장됩니다.
  6. return a + b가 실행되어 8을 함수 밖으로 반환합니다.
  7. result 변수에 반환값 8이 저장됩니다.
실습: function.html 실습: function_1.html

3. 함수 표현식

개념 정의
함수 표현식은 함수를 값처럼 만들고, 그 함수를 변수 또는 상수에 저장하는 작성 방식입니다.
작성 문법 
const 상수명 = function() {
  실행할 코드;
};

상수명();
함수 표현식에서는 보통 const를 사용합니다. 함수가 저장된 상수명에 다른 값을 다시 대입하지 않기 위해서입니다.
이름이 있는 함수 표현식 
const 상수명 = function 함수명() {
  실행할 코드;
};
이름이 없는 함수 표현식 
const 상수명 = function() {
  실행할 코드;
};

4. 화살표 함수

개념 정의
화살표 함수는 => 기호를 사용해서 함수를 작성하는 문법입니다.
기본 작성 문법 
const 상수명 = () => {
  실행할 코드;
};

상수명();
매개변수가 있는 화살표 함수 
const 상수명 = (매개변수1, 매개변수2) => {
  실행할 코드;
  return 반환값;
};

상수명(인수1, 인수2);
실행 코드가 return 한 줄이면 중괄호와 return 생략 가능 
const add = (a, b) => a + b;
중괄호 {}를 작성하면 return을 직접 작성해야 합니다.
중괄호 {}를 생략하고 식 하나만 작성하면 그 식의 결과가 자동으로 반환됩니다.
실습: arrow_func.html 실습: arrow_function.html

5. 매개변수와 인수

개념 정의
매개변수는 함수가 값을 받기 위해 선언하는 변수입니다.
인수는 함수를 호출할 때 전달하는 실제 값입니다.
용어위치설명
매개변수함수 선언부의 소괄호함수가 값을 받기 위해 선언한 변수
인수함수 호출부의 소괄호함수를 호출할 때 전달하는 실제 값
작성 문법 
function 함수명(매개변수1, 매개변수2) {
  실행할 코드;
}

함수명(인수1, 인수2);
코드 읽는 순서 
function sum(num1, num2) {
  return num1 + num2;
}

sum(10, 20);
  1. num1, num2는 매개변수입니다.
  2. 10, 20은 인수입니다.
  3. 첫 번째 인수 10은 첫 번째 매개변수 num1에 저장됩니다.
  4. 두 번째 인수 20은 두 번째 매개변수 num2에 저장됩니다.
실습: gugudan.html 실습: prameter_default.html

6. 매개변수의 특징과 기본값

개념 정의
함수 호출 시 인수를 전달하지 않으면 해당 매개변수에는 undefined가 저장됩니다.
확인 코드 
function print(a, b) {
  console.log(a, b);
}

print();
print()처럼 인수 없이 호출하면 a와 b에는 undefined가 저장됩니다.
기본값 작성 문법 
function 함수명(매개변수1 = 기본값1, 매개변수2 = 기본값2) {
  실행할 코드;
}
기본값 코드 읽기 
function sum(a = 10, b = 10) {
  console.log(a, b);
}

sum();
sum(20, 30);
  1. sum()은 인수를 전달하지 않습니다.
  2. a는 기본값 10을 사용합니다.
  3. b는 기본값 10을 사용합니다.
  4. sum(20, 30)은 인수를 전달합니다.
  5. 전달된 인수가 있으면 기본값 대신 인수 값이 사용됩니다.
실습: function_1.html 실습: prameter_default.html

7. return

개념 정의
return은 함수 실행 결과를 함수 밖으로 반환하는 문법입니다.
작성 문법 
function 함수명() {
  return 반환값;
}
코드 읽는 순서 
function add() {
  return 10 + 20;
}

let result = add();
console.log(result);
  1. add()를 만나면 함수가 호출됩니다.
  2. 함수 안에서 10 + 20이 계산됩니다.
  3. return이 계산 결과 30을 함수 밖으로 반환합니다.
  4. result 변수에 30이 저장됩니다.
  5. console.log(result)가 30을 콘솔에 출력합니다.
자주 헷갈리는 부분
return은 값을 출력하지 않습니다. 값을 함수 밖으로 반환합니다.
console.log()는 값을 콘솔에 출력해서 확인합니다.
실습: add_func.html

8. return으로 함수 종료하기

개념 정의
return;처럼 반환값 없이 작성하면 함수를 즉시 종료합니다.
작성 문법 
function 함수명() {
  if (조건식) {
    return;
  }

  실행할 코드;
}
코드 읽는 순서 
function sum(num1, num2) {
  if (typeof num1 !== "number" || typeof num2 !== "number") {
    return;
  }

  return num1 + num2;
}
  1. 함수가 호출됩니다.
  2. 두 인수 중 하나라도 숫자가 아니면 return;이 실행됩니다.
  3. return;은 함수를 즉시 종료합니다.
  4. 숫자 두 개가 전달되면 return num1 + num2가 실행됩니다.
실습: return_exit.html

9. 스코프

개념 정의
스코프는 변수나 함수를 사용할 수 있는 코드 범위입니다.
구분설명
전역 스코프함수나 블록 바깥 영역입니다. 여러 위치에서 접근할 수 있습니다.
지역 스코프함수나 블록 내부 영역입니다. 보통 해당 영역 안에서만 접근할 수 있습니다.
함수 스코프함수의 중괄호 {}를 기준으로 범위가 나뉩니다.
블록 스코프if, for, 단독 블록의 중괄호 {}를 기준으로 범위가 나뉩니다.
작성 문법 
let 전역변수 = 값;

function 함수명() {
  let 지역변수 = 값;
}
let, const는 블록 스코프를 따릅니다.
var는 블록 스코프를 따르지 않고 함수 스코프를 따릅니다.
실습: global.html 실습: block_scope.html 실습: block_scope2.html 실습: reference.html

10. 호이스팅

개념 정의
호이스팅은 코드 실행 전에 선언 정보를 먼저 준비하는 동작입니다.
함수 선언문은 선언 위치보다 위에서 호출할 수 있습니다.
함수 표현식과 화살표 함수는 상수에 함수가 저장된 후에 호출해야 합니다.
함수 선언문 
printHello();

function printHello() {
  console.log("Hello");
}
위 코드는 실행됩니다. 함수 선언문은 실행 전에 먼저 준비됩니다.
함수 표현식 
printHello();

const printHello = function() {
  console.log("Hello");
};
위 코드는 오류가 발생합니다. printHello 상수에 함수가 저장되기 전에 호출했기 때문입니다.
실습: hoisting_func.html 실습: hoisting_express.html

11. 즉시 실행 함수

개념 정의
즉시 실행 함수는 함수를 선언하는 동시에 바로 호출해서 실행하는 함수입니다.
작성 문법 
(function() {
  실행할 코드;
})();
코드 읽는 순서 
(function init() {
  console.log("initialized");
})();
  1. function init() { ... } 형태의 함수가 작성됩니다.
  2. 함수 전체를 소괄호로 감싸 함수 표현식으로 만듭니다.
  3. 마지막 ()가 붙어 즉시 호출됩니다.
  4. 한 번 실행된 뒤에는 함수명으로 다시 호출할 수 없습니다.
실습: llFE.html
